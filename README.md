# NTFSDesk
一个简单玩玩自己用的NTFS用的小玩具，新手 随便玩玩（大佬轻点喷）

## 功能

- 显示当前已挂载的磁盘/分区
- 查看容量、文件系统、挂载路径、读写状态
- 打开卷到 Finder
- 推出磁盘
- NTFS 写入开关界面状态
- 浏览文件夹
- 识别常见文件类型：图片、视频、音频、PDF、文本、代码、文档、压缩包、镜像、应用包
- 快速预览文本/代码/部分文件信息


## 说明
有一个zip解压双击就能用 试试水 插U盘好使点
当前 NTFS 写入按钮是应用内开关状态。真正控制 NTFS 读写挂载需要接入 macFUSE/ntfs-3g 或单独的特权 helper。
